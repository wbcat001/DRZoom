# 4. 分析画面のUI/UX
高次元データに対して、次元削減とクラスタリングの木構造を用いて、インタラクションを提供することで、データの特徴と次元削減手法の効果を探索的に分析することを目的とする

## 分析画面の種類
- 次元削減ビュー
- デンドログラムビュー
- パラメータ選択ビュー

## 可視化エンコーディングの詳細
データを可視化する際にどうエンコーディングするかの詳細

### 次元削減ビュー
高次元データを次元削減した2次元上の座標を可視化する
- data: (index, x, y, label)
- position: 2次元に次元削減した点をプロットする
- mode: データ選択、範囲選択、ズームの切り替え、toggleで切り替えする
オプション
- color: データに付いているラベル or クラスタリング結果のラベル or 同一色。
- size: 2 point
- shape: circle(marker)


### デンドログラムビュー
高次元データをクラスタリングして得られるクラスタの階層構造を可視化する
- data: linkage matrix(child1, child2, parent, dist, size, something score), 
- position: linkage matrixをもとに計算した座標
オプション
- color: 青、強調表示は黃
- width: 1 or クラスタサイズ: toggleで切り替えする

### 詳細情報ビュー
インタラクションに応じて、詳細な情報を提供する
