## API 仕様

### エンドポイント
```
POST /api/zoom/redraw
```

### リクエスト
```json
{
  "point_ids": [0, 1, 2, ...],
  "dr_method": "umap",
  "n_neighbors": 15,
  "min_dist": 0.1,
  "n_epochs": 200
}
```

### レスポンス（成功）
```json
{
  "status": "success",
  "coordinates": "base64エンコード",
  "shape": [N, 2],
  "point_ids": [0, 1, 2, ...]
}
```

### レスポンス（エラー）
```json
{
  "status": "error",
  "message": "エラー説明",
  "coordinates": null,
  "shape": null,
  "point_ids": null
}
```
